# Motion Graphics Production via Structural Definition  
## ～ A Generative Approach with After Effects and Python ～

This repository presents a **theoretical framework for automating motion graphics using Bezier structure definitions and Python** in Adobe After Effects.

🎓 Note: This repository does **not** include implementation code or demo files.  
Its primary focus is on documenting and sharing the **concept and methodology**.

👉 For detailed proposals and background theory, see [`ae_python_structured_motion_en.md`](./ae_python_structured_motion_en.md).
